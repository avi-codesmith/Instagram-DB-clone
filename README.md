# 📸 Instagram Database Clone (MySQL)

A relational database project that recreates the core database structure of Instagram using **MySQL**.

🚀 Features

* 👤 User accounts
* ❤️ Likes system
* 💬 Comments
* 🤝 Follow & Unfollow relationships
* 🏷️ Hashtags
* 🔖 Saved posts
* 🗄️ Well-structured relational schema
* 🔑 Primary Keys & Foreign Keys
* ⚡ Optimized SQL queries

---

## 🛠️ Tech Stack

* **Database:** MySQL
* **Language:** SQL
* **Tool:** MySQL Workbench + CLI

---

## 📂 Database Schema

The database includes tables such as:

* Users
* Posts
* Comments
* Likes
* Followers
* Unfollow data
* Hashtags
* Post_Hashtags
* Saved_Posts

Each table is connected using proper foreign key relationships to maintain data integrity.
